✓ Downloaded — next steps
  1. On your media NAS, create a folder for the agent, e.g. /volume1/docker/plex-agent/ — the volume is wherever you keep your Docker projects
  2. Download agent.py and requirements.txt from the GitHub repo and place them in plex-agent/app/
  3. Unzip plex-agent-setup.zip and copy docker-compose.yml into plex-agent/ and config/config.yaml into plex-agent/config/
  4. Open Container Manager on your media NAS → Project → Create → set the path to your plex-agent folder
  5. Click Build — the agent will start watching and announce its paths to this receiver automatically
  6. Return to the main UI to configure path mappings
